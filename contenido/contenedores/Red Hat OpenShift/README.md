# Red Hat OpenShift

![OpenShift](../../../images/section_openshift_logo.png)

En esta sección podrás encontrar todo el contenido relacionado a Red Hat OpenShift (RHOS).

Red Hat OpenShift es una plataforma de aplicaciones de contenedores de código abierto basada en el proyecto abierto OKD (Open Kubernetes Distribution) para el desarrollo y la implementación de aplicaciones empresariales.

Para más información, consulta [el sitio oficial de OpenShift](https://www.openshift.com/).
